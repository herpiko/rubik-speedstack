rubik-speedstack
================

speedstack timer for playing 333 rubik's cube, including generated scramble notation

###dependency
```$ npm install -d```

###basic usage

```$ node speedstack.js```

- space : start & stop timer
- c : clear session
- s : toggle scramble for next race
- q : quit
